#  Quiz System (JavaFX)

##  Overview
A desktop-based quiz application built using JavaFX.

##  Features
- Secure login system  
- Interactive quiz interface  
- Automatic score calculation  
- Logout functionality  

##  Technologies Used
- Java  
- JavaFX  
- IntelliJ IDEA  

##  How to Run
1. Open project in IntelliJ IDEA  
2. Run the Main class  
3. Start quiz system
